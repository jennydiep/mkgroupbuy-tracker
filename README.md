# Set up

## Install Node JS
## Install the following:

    npm install -g @angular/cli  # to install angular
    npm install -g ionic # to install ionic

## Install dependencies
    cd mkgroupbuy-tracker
    npm install

## Run app within mkgroupbuy-tracker folder
    ionic serve
